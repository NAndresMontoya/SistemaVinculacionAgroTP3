 # Sistema de Vinculación Agro

Proyecto académico desarrollado en Java para la materia Seminario de Práctica Informática.

## Descripción

El sistema permite vincular productores agrícolas con mecánicos especializados para resolver problemas relacionados con maquinaria agrícola.

## Funcionalidades

- Registro de productores
- Registro de mecánicos
- Creación de solicitudes de reparación
- Visualización de solicitudes
- Aceptación de solicitudes por parte de mecánicos
- Consulta de datos de contacto del productor

## Conceptos de Programación Orientada a Objetos aplicados

### Encapsulamiento
Los atributos de las clases son privados y se acceden mediante métodos.

### Herencia
Las clases Productor y Mecanico heredan en la clase Usuario.

### Objetos
Se crean objetos de tipo Usuario, Productor, Mecanico, Maquinaria y Solicitud.

### Constructores
Las clases utilizan constructores para inicializar sus atributos.

## Tecnologías utilizadas

- Java
- Swing
- Programación Orientada a Objetos (POO)
